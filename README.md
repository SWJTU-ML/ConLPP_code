This is the original implementation of the algorithm CLPP in the paper "Linear dimensionality reduction method based on topological properties" (
[DOI](https://doi.org/10.1016/j.ins.2022.12.098) ;
[Direct Link](https://www.sciencedirect.com/science/article/pii/S0020025522015936)
).

## Requirements

Current environment:

* python == 3.9
* scipy == 1.6.3
* scikit-learn == 0.24.2
* networkx == 2.5.1
